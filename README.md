# Streamify

Streamify is a learning language platform that allows users to engage with their community through chat and video calls. It allows users to connect, interact, and build meaningful relationships within their community through real-time messaging and video calls.

**Live** : [https://streamify.ucokman.web.id](https://streamify.ucokman.web.id).

## Feature

- Community Interaction
- Realtime Chat and Video Call

## Technology Used

- [React JS](https://react.dev)
- [Tailwind CSS](https://tailwindcss.com)
- [Tanstack](https://tanstack.com)
- [Getstream.io](https://getstream.io/)
- [Mongodb](https://www.mongodb.com/)
- [Golang](https://go.dev/)
